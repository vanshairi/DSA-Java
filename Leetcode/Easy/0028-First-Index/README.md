# 28. Find the Index of the First Occurrence in a String

## Problem
Given two strings `haystack` and `needle`, return the index of the first occurrence of `needle` in `haystack`. If `needle` is not part of `haystack`, return `-1`.

## Difficulty
Easy

## Approach
- Use Java's built-in `indexOf()` method.
- Search for the first occurrence of `needle` in `haystack`.
- Return the index if found; otherwise, `indexOf()` returns `-1`.

## Time Complexity
O(n × m) *(implementation-dependent)*

## Space Complexity
O(1)
