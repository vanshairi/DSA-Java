# 1108. Defanging an IP Address

## Problem
Given a valid IPv4 address, return its defanged version by replacing every `"."` with `"[.]"`.

## Difficulty
Easy

## Approach
- Use Java's built-in `replace()` method.
- Replace every occurrence of `"."` with `"[.]"`.
- Return the modified string.

## Time Complexity
O(n)

## Space Complexity
O(n)
