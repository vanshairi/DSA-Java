# 1480. Running Sum of 1d Array

## Problem
Given an array `nums`, return the running sum of the array, where each element is the sum of all previous elements including itself.

## Difficulty
Easy

## Approach
- Create a new array to store the running sums.
- Traverse the input array while maintaining a cumulative sum.
- Store the cumulative sum at each index.
- Return the resulting array.

## Time Complexity
O(n)

## Space Complexity
O(n)
